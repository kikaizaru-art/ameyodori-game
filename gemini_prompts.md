# 雨宿りの回廊 — Gemini ドット絵生成プロンプト

## 共通指示（全プロンプトの冒頭に付ける）
```
以下の指示に従い、ゲーム用ドット絵を1枚生成してください。
- スタイル: レトロゲーム風ピクセルアート
- パレット: 暗い青紫ベース（夜の雰囲気）、8〜12色以内
- 統一カラー: #0a0a12(背景暗), #1a1a2e(床暗), #2a2a3e(壁), #ffdd88(暖光), #e08040(姉), #4488cc(弟), #661133(敵目), #08000f(影)
```

## バッチ1: マップタイル（32×32px）

### 1. floor（床）
```
32×32ピクセルのドット絵。雨の夜の商店街の石畳の床。
色味: 暗い青紫(#1a1a2e)ベース、わずかに濡れた反射。
タイルとしてシームレスに繰り返せるデザイン。背景なし（単色塗りつぶし）。
```

### 2. wall（壁）
```
32×32ピクセルのドット絵。商店街の壁/閉まったシャッター。
色味: やや明るい青灰色(#2a2a3e〜#333350)。レンガかコンクリートの質感。
タイルとして並べた時に壁に見えるデザイン。
```

### 3. shop（商店）
```
32×32ピクセルのドット絵。閉店した小さな店舗の正面。
色味: 暗い紫系(#2e2438〜#3a2e48)。シャッターが下りている。
小さな看板や窓の痕跡があると雰囲気が出る。
```

### 4. streetlight（街灯）
```
32×32ピクセルのドット絵。商店街の街灯。上部に暖かい黄色(#ffdd88)の光。
支柱は灰色(#555566)。光の部分が明るく輝いている。
背景は暗い床色(#1a1a2e)。
```

### 5. mirror（ミラー）
```
32×32ピクセルのドット絵。壁に取り付けられた反射ミラー。
青白い鏡面(#88aadd)が特徴的。フレームは金属的な色。
背景は暗い床色。
```

### 6. goal（出口）
```
32×32ピクセルのドット絵。商店街の出口/ゲート。
上方向に暖かい光(#ddcc88〜#eedd99)が漏れている。
希望を感じさせる暖色。周囲より明るい。
```

## バッチ2: ギミックタイル（32×32px）

### 7. pushbox（押せる箱）
```
32×32ピクセルのドット絵。木製の箱/クレート。
茶色系(#8B6914〜#A0522D)で木目模様。
押せることが分かるように矢印マークか取っ手を描く。
背景は透過。
```

### 8. breaklight（壊れかけの街灯）
```
32×32ピクセルのドット絵。ヒビの入った壊れかけの街灯。
通常の街灯と似ているが、光がオレンジ〜赤みがかっている。
ガラス部分にヒビが入っており、不安定さを表現。
背景は暗い床色。
```

### 9. switch_off（スイッチOFF）
```
32×32ピクセルのドット絵。床に埋め込まれた丸いスイッチ/ボタン。
OFF状態: 暗い灰色〜暗赤色。踏めることが分かるデザイン。
床タイルの上に乗っている感じ。
```

### 10. switch_on（スイッチON）
```
32×32ピクセルのドット絵。床に埋め込まれた丸いスイッチ/ボタン。
ON状態: 明るい緑色(#44cc66)に光っている。
switch_offと同じ形だが色が変わったもの。
```

### 11. door（扉/ゲート）
```
32×32ピクセルのドット絵。通路を塞ぐ鉄格子の扉。
暗い金属色。鍵マークか錠前が見える。
壁とは明らかに違う、「開けられそう」な見た目。
```

### 12. match_item（マッチアイテム）
```
32×32ピクセルのドット絵。地面に落ちているマッチ箱。
赤い箱に小さな炎のアイコン。光を感じさせる暖色(#ff9944)。
拾えるアイテムだと分かるように少し光っている。
背景は透過。
```

### 13. sister（姉）
```
32×32ピクセルのドット絵。正面向きの少女キャラクター。
2.5頭身のデフォルメ。活発な印象。
髪: オレンジ〜茶色。服: 暖色系(#e08040, #cc6633)。
大きな目で表情がわかる。背景は透過。
```

### 14. brother（弟）
```
32×32ピクセルのドット絵。正面向きの小さな男の子キャラクター。
2頭身のデフォルメ。姉より一回り小さい。臆病そうな表情。
髪: 暗めの青。服: 青系(#4488cc, #3366aa)。
姉に寄り添うような雰囲気。背景は透過。
```

## バッチ4: 敵（64×64px、透過背景）

### 15. shadow_enemy（影の敵）
```
64×64ピクセルのドット絵。黒い人型の影/シルエット。
不定形で輪郭がゆらめいている。ホラー的な存在感。
体: 黒〜極暗紫(#08000f〜#110022)。
目: 赤く光る2つの点(#661133〜#cc2244)。
腕が不自然に長く、全体的に歪んだ人型。
背景は透過。恐怖を感じさせるが、グロテスクではない。
```

---

## 使い方

1. Geminiを開く
2. 「共通指示」を最初に伝える
3. バッチごとにプロンプトをコピペして画像を生成
4. 生成された画像を32×32（または64×64）にリサイズしてPNGで保存
5. ゲームのtextures.jsをアセット読み込みに差し替え
